要用OpenCV写一个识别红色三角形和蓝色圆形的小程序。
但是我啥都不会，所以我选择了看学长发的那本书，看了不少，但是没学到多少，就不看了。于是我就去BB大学随便找了一个教程看，结果讲得一点都不好，看了几节也不看了，又另外找了一个印度人讲的，讲得确实好，不过口音有点难受，然后我今天上程序设计课的时候等不及了，就自己照着上节课的文档试着做了一下，结果做好了。我觉得我有个学长说的很对，你要是想进步，就去做项目，去做你不会做的，才能学会你不会的。


这个程序的思路就是先存下来群里发的那张图片，然后自己定义一个因变函数，再创造四个滚动条方便调参，分别对红色与蓝色进行阈值化，分别绘制轮廓，然后打出框框与标签，就好了。

不过也有一些问题，我有的代码是直接从上节课的文档复制的，带着cv::这种命名空间，可是我已经使用了using namespace cv;了，不知道为什么把这些cv::删掉会报错，一会我再问问学长。
